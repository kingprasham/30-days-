========================================
CUSTOMER TRACKING & BILLING MANAGEMENT SYSTEM
INSTALLATION GUIDE
========================================

STEP 1: VERIFY FILES
--------------------
Make sure all files are in: C:\xampp\htdocs\papa\30 days\

STEP 2: START XAMPP
-------------------
1. Open XAMPP Control Panel
2. Start "Apache"
3. Start "MySQL"

STEP 3: SETUP DATABASE
----------------------
1. Open your web browser
2. Go to: http://localhost/papa/30%20days/setup_database.php
3. Wait for setup to complete
4. You should see "Database setup completed successfully!"

STEP 4: LOGIN
-------------
1. Go to: http://localhost/papa/30%20days/
2. Login with:
   Username: admin
   Password: 123456
3. IMPORTANT: Change password after first login!

STEP 5: START USING
-------------------
1. Dashboard - View all statistics and charts
2. Upload Excel - Upload your Customer new.xlsx file
3. Customers - Manage customers
4. Dealers - Add dealer information
5. Products - View/manage products (auto-populated from Excel)
6. Reports - View various reports

========================================
TROUBLESHOOTING
========================================

Problem: "Database connection failed"
Solution: Make sure MySQL is running in XAMPP

Problem: "Cannot find database setup file"
Solution: Make sure setup.sql is in the database folder

Problem: Excel upload not working
Solution: Check that uploads/excel folder exists and has write permissions

Problem: Charts not showing
Solution: Make sure you have internet connection (for CDN libraries)
